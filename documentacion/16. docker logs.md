## Docker logs

### Ver los logs de un contenedor
```bash
docker logs apache
```
Por defecto el driver del log es `json-log`

Los logs se almacenan en `/var/lib/docker/containers/contenedor/contenedor-json.log`


### Asignar configuracion de log a un contenedor
```bash
docker run -d --name apache -p 80:80 --log-driver json-file --log-opt max-size=10m --log-opt max-file=3 httpd
```
- **--log-driver** Le asigno un log driver (json es el por defecto).
- **--log-opt max-size** Máximo tamaño de log que va a guardar antes de que rote.
- **--log-opt max-file=3** Máximo de archivos rotados guardados para evitar llenado de disco. Conseva los ultimos 3 archivos rotados.

Las opciones que hyaa disponibles dependen del tipo de driver

Podemos asignar una configuracion global a todos los contenedores por defecto añadiendo la config en el `daemon.json`

Podemos usar drivers como logstash, fluentd, journalctl... para enviar allí el log
